# Project Flux

Framework for simulating the Data Engineering lifecycle.

## Vision

Actually, the data engineering lifecycle, as described by the excelent book, Fundamentals of Data Engineering, encompasses a 5 stage process:

1. Generation
2. Ingestion
3. Storage
4. Transformation
5. Serving

In this project, we aim to build a simulated environment, and reference code, to make a end-to-end data flux to demonstrate this lifecycle and the concepts behind this.
